## 📌 About Repository

This repository contains **Web Technology Lab Programs** designed to demonstrate the fundamentals of modern web development.

💡 It helps learners understand how to:
- Structure web pages using **HTML**
- Style designs using **CSS**
- Add interactivity using **JavaScript**
- Build responsive layouts with **Bootstrap**
- Handle backend logic using **PHP**

---

## 📚 List of Programs

---

### 1️⃣ 🧾 Personal Details using HTML Text Formatting
A simple webpage displaying personal information like name, age, address, and hobbies.

**Concepts Used**
- `<b>`, `<i>`, `<u>`
- `<strong>`, `<em>`

🎯 *Focus: HTML text formatting*

---

### 2️⃣ 🖼️ Webpage with Image, Hyperlink & Nested Table
Includes image, hyperlink, and nested table structure.

🎯 *Focus: HTML multimedia & table design*

---

### 3️⃣ 📝 HTML Form Creation
User input form with fields like name, email, gender, and date of birth.

🎯 *Focus: Form elements & input handling*

---

### 4️⃣ 🎨 CSS Styling (Inline, Internal & External)
Demonstrates all three CSS techniques.

🎯 *Focus: Styling methods*

---

### 5️⃣ 📱 Responsive Webpage using Bootstrap
Creates mobile-friendly responsive layouts.

🎯 *Focus: Responsive design*

---

### 6️⃣ ⚡ JavaScript Form Validation
Validates user input and prevents incorrect data submission.

🎯 *Focus: Data validation*

---

### 7️⃣ 🔄 Dynamic Content using JavaScript DOM
Manipulates webpage content dynamically.

🎯 *Focus: DOM operations*

---

### 8️⃣ 🧮 Simple Calculator using JavaScript
Performs basic arithmetic operations.

🎯 *Focus: JavaScript logic*

---

### 9️⃣ 🐘 PHP Welcome Page with Date & Time
Displays welcome message with current date and time.

🎯 *Focus: Basic PHP scripting*

---

### 🔟 📥 PHP Form Handling (POST Method)
Processes form data securely using POST method.

🎯 *Focus: Backend form handling*

---

### 1️⃣1️⃣ 🔤 PHP String Manipulation

🎯 *Focus: PHP string functions*

🎯 *Focus: Data structures in PHP*
